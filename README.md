# Daily Reflection Tree – DT Assignment

## Overview

This project is a deterministic reflection tool that helps users analyze their day using a structured decision tree.

No AI/LLM is used at runtime. All logic is predefined.

---

## Axes Covered

1. Locus of Control (Victim vs Victor)
2. Contribution vs Entitlement
3. Radius of Concern (Self vs Others)

---

## How It Works

* User selects options
* System follows fixed paths
* Reflections are shown
* Ends with summary
